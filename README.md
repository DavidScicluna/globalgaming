# Global Gaming Application

Built with [React JS](https://reactjs.org) & [Redux](https://redux.js.org).

## Features
  1. Registration
  2. Search Movies/TV Shows
  3. Sort by genre
  4. Like Movies/TV Shows
  5. Add Movies/TV Shows to watchlist
  6. Customize
